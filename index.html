VGA_PAINT_TOOL_FPGA

A real-time paint application built on FPGA, using VGA output and digital inputs to draw, erase, and move a color box across the screen. Controlled via physical FPGA input pins for color selection. Designed with FSM logic and pixel mapping in Verilog. Compatible with Nexys A7 and PYNQ FPGA boards.

RISC_V_PIPELINED_CPU

A fully pipelined 5-stage RISC-V CPU written in Verilog. Supports instruction fetch, decode, execute, memory access, and write-back stages with hazard handling. Includes assembler for R, I, B, S, J instructions. Branch prediction and hazard mitigation tested.

RASPBERRY_PIHOLE_SERVER

A network-level ad-blocking DNS server deployed on a Raspberry Pi to filter ads and trackers across all connected devices. Blocks ads on all platforms with no browser extensions needed Enhances privacy, reduces bandwidth and speeds up browsing

SELF_BALANCING_ROBOT

A two-wheeled autonomous robot that uses feedback from an MPU6050 sensor to maintain balance via PID control. Smooth PID tuning for fast, stable correction. Real-time angle estimation with accelerometer + gyro fusion. Built with Arduino, L298N motor driver, and custom chassis.
